# 前提
私は、日々の情報収集にDiscordを活用しています。
具体的には、ニュースや最新情報を各種RSSフィードから取得し、Discordに送信されるようにしています。

しかし、Discordに送信されるニュースに特にフィルタをかけていないため、すべてが気になる情報ではありません。
リンクを開くことなくスルーする記事がほとんどです。

その中で、気になる記事や読んでみて参考になった記事などを再度開こうとすると、Discord上では埋もれてしまい、ブラウザの履歴を遡るぐらいしか方法がありませんでした。

# 概要
本リポジトリでは、Discord上にある任意の記事をNotionデータベースに蓄積する仕組みを構築しました。

具体的には、Discord上で⭐のリアクションが押された場合に、その記事の「タイトル」「URL」「チャンネル名」「投稿日時」がNotionデータベース上に記録されるようにしています。

# 手順
改めて、Discordを活用した情報収集の流れを説明します。

1. RSSフィードからニュースや最新情報をDiscordに集約
2. Discord上で気になる記事・参考になった記事に⭐でリアクション
3. 1時間に1度Discord Botが起動し、Discord上で⭐リアクションが押された投稿を抽出
4. Notionデータベースに更新されていない投稿でフィルタリングし、「タイトル」「URL」「チャンネル名」「投稿日時」をNotionデータベースに記録

このリポジトリでは、手順3,4を実行します。

# 使用ツール
- Discord
- Discord Bot
- Notion
- Github Actions
